# FlowFinance — Контекст проекта для Claude Code

## Что это за проект

**FlowFinance** — персональная финансовая учётная система с акцентом на прогнозирование денежных потоков и предсказание кассовых разрывов.

Основная идея: пользователь видит в формате календаря все фактические и запланированные транзакции, а система автоматически рассчитывает будущий баланс и предупреждает о дате, когда деньги могут закончиться.

## Стек технологий

- **React 19 + TypeScript 5** — фронтенд
- **Vite 6** — сборщик, dev-сервер на порту 3000
- **Tailwind CSS (CDN)** — стилизация
- **Recharts** — графики (прогноз баланса)
- **Нет бэкенда** — пока всё на моковых данных в `data/mockData.ts`
- Переменная `GEMINI_API_KEY` в `.env.local` — зарезервирована, пока не используется

## Архитектура

```
App.tsx                 ← глобальное состояние: currentDate, selectedDate, transactions
├── Header              ← навигация по месяцам
├── LeftSidebar         ← счета, бюджеты, итоги месяца
├── MainCalendar        ← основной календарь с транзакциями (drag-and-drop)
├── RightSidebar        ← детали выбранного дня (транзакции / бюджет)
└── AnalyticsPanel      ← линейный график прогноза баланса на 60+ дней
```

## Ключевые сущности (types.ts)

| Тип | Поля |
|-----|------|
| `Transaction` | id, date, description, amount (+доход / -расход), type: `actual`\|`planned`, categoryId, accountId |
| `Account` | id, name, type: checking/savings/cash/credit, balance |
| `Category` | id, name, type: income/expense, color (Tailwind-класс) |
| `Budget` | categoryId, amount (лимит в месяц) |

## Основная логика

- **Прогнозируемый баланс на дату** = сумма балансов всех счетов + сумма всех транзакций до этой даты (`calculateProjectedBalanceForDate` в `utils/helpers.ts`)
- **Кассовый разрыв** — момент, когда прогнозируемый баланс уходит в минус или ниже порога
- Транзакции `planned` — будущие/запланированные (отображаются пунктирной рамкой)
- Транзакции `actual` — уже проведённые

## Дизайн-система

- macOS/glassmorphism стиль: `backdrop-blur-xl`, полупрозрачные фоны
- Поддержка тёмной темы через `dark:` классы Tailwind
- Левая панель — 300px, правая — 350px, центр — растягивается
- Цветовые сигналы баланса: зелёный (норма) → жёлтый (<$5000) → красный (минус)

## Запуск


## Направление развития

Проект задуман как инструмент персонального финансового планирования:
1. Учёт кредитов и регулярных платежей
2. Прогнозирование платежей на будущие периоды
3. Предсказание кассовых разрывов (когда не хватит денег)
4. Возможная интеграция с AI (Gemini API) для умных подсказок
